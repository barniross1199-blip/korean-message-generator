# Korean Message Generator

Мобильное React + Vite + Tailwind приложение для русскоязычных иностранцев в Корее. Оно помогает быстро создавать корейские сообщения для работы, продавцов и бытовых ситуаций.

## Возможности

 codex/create-korean-message-generator-app
- Режим «Готовые шаблоны»: 6 категорий — Работа, Зарплата, Продавец, Доставка, Жильё, Больница.
- По 5 готовых сценариев в каждой категории.
- Для каждого сценария есть корейский текст, русское значение, романизация и уровень вежливости.
- Кнопки для копирования корейского текста, более мягкой формулировки и более прямого варианта.
- Режим «Своё сообщение»: пользователь вводит русский текст, выбирает ситуацию и стиль, а демо-логика формирует корейский вариант без AI API.

## Локальный запуск

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Запустите dev-сервер:

   ```bash
   npm run dev
   ```

3. Откройте адрес из терминала, обычно `http://localhost:5173`.

## Сборка

```bash
npm run build
```
